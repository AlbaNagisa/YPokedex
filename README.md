## Installation

Clonner le repo :

```sh
git clone https://github.com/AlbaNagisa/YPokedex
```

Acceder au repertoire creer

```sh
cd YPokedex
```

Installer les dependences

```sh
npm install
```

Lancer le projet en developpement

```sh
npm run dev
```

Lancer le projet en production

```sh
npm run build
npm run start
```
